== Description ==

WC SpeedUp הוא פלאגין אופטימיזציה מקיף לאתרי WooCommerce שעוזר לך:

= אבחון ביצועים =
* בדיקת תצורת השרת (PHP, MySQL, OPcache)
* בדיקת תצורת וורדפרס
* בדיקת תצורת WooCommerce
* ניתוח מסד הנתונים
* זיהוי פלאגינים כבדים
* בדיקת מצב המטמון

= ניקוי מסד נתונים =
* מחיקת גרסאות ישנות של פוסטים
* ניקוי טיוטות אוטומטיות
* מחיקת פריטים בפח
* ניקוי תגובות ספאם
* מחיקת טרנזיינטים שפג תוקפם
* ניקוי מטא-דאטה יתום
* אופטימיזציית טבלאות

= אופטימיזציות WooCommerce =
* השבתת Cart Fragments ב-AJAX
* הסרת סגנונות WooCommerce מדפים שלא צריכים אותם
* הגבלת מספר הוריאציות
* השבתת מד חוזק סיסמה
* הגבלת מוצרים קשורים
* השבתת WooCommerce Admin
* השבתת Marketing Hub

= אופטימיזציות כלליות =
* השבתת אימוג'ים
* השבתת oEmbed
* הגבלת גרסאות פוסטים
* אופטימיזציית Heartbeat API
* השבתת XML-RPC
* Lazy Loading לתמונות
* הסרת Query Strings מקבצים סטטיים
* Defer ל-JavaScript
* DNS Prefetch
* Preconnect

= תכונות נוספות =
* דשבורד מרכזי עם ציון ביצועים
* המלצות מותאמות אישית
* ניקוי אוטומטי יומי
* תמיכה ב-RTL

== Installation ==

1. העלה את תיקיית `wc-speedup` לתיקיית `/wp-content/plugins/`
2. הפעל את הפלאגין דרך תפריט 'פלאגינים' בוורדפרס
3. עבור ל-'WC SpeedUp' בתפריט הניהול
4. הפעל את הרחבות הדרושות בדף ההגדרות

== Frequently Asked Questions ==

= האם הפלאגין בטוח לשימוש? =

כן, כל הפעולות ניתנות לביטול והפלאגין לא משנה קבצי ליבה.

= האם צריך לגבות לפני ניקוי מסד הנתונים? =

מומלץ תמיד לגבות את מסד הנתונים לפני פעולות ניקוי.

= האם הפלאגין עובד עם פלאגיני מטמון אחרים? =

כן, הפלאגין משלים פלאגיני מטמון קיימים ולא מחליף אותם.
